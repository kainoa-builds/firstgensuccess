# FirstGen Success

A free, no-BS playbook helping first-generation college students — and their parents — navigate college, from financial aid to their first internship.

FirstGen Success demystifies the unwritten rules of college for students who are the first in their family to go: money and financial aid, study skills, and especially **career success** (resumes, internships, networking, and job-search tools like LinkedIn, Handshake, and Indeed). It also speaks to parents who never went to college themselves, so families can learn the playbook together.

## Pages

| File | What it is |
|------|------------|
| `FirstGen Success.html` | Home — the pitch, "the problem nobody names," the Playbook, Essentials, Parents, and About |
| `Tools.html` | The career toolkit — platforms to get on today |
| `styles.css` | Shared styles for every page |

## Running it

It's a static site with no build step. Open `FirstGen Success.html` in a browser, or serve the folder:

```bash
python3 -m http.server
```

Then visit `http://localhost:8000/FirstGen%20Success.html`.

## Deploying

Any static host works — GitHub Pages, Netlify, or Vercel. Point it at the project root; no configuration needed.
